Результаты гидродинамических исследований скважин

Программный комплекс РИС (Результаты Исследования Скважин) предназначен для ввода и обработки данных по всем видам гидродинамических исследований, которые проводятся на скважинах. Область применения комплекса — нефтегазодобывающая промышленность, предприятия, непосредственно осуществляющие эксплуатацию нефтяных и нагнетательных скважин.

Рабочие места программы устанавливаются в бригадах по исследованию скважин (ЦНИПР), которые непосредственно проводят гидродинамические исследования, а также на рабочих местах специалистов нефтепромыслов — технологической и геологической служб. Программный комплекс поделен на две части. Первая часть - ввод и редактирование данных. Вторая часть генератор запросов и отчетов. Справочная часть программы содержит всю информацию о фонде скважин с учетом административной привязки (НГДУ, ЦДНГ, куст, месторождение, бригада добычи).

Справочник по скважинам может быть привязан к существующим базам данных нефтегазодобывающего предприятия или иметь свою систему ввода данных по скважине. Система ввода данных по результатам исследования имеет два варианта ввода информации. Первый — ручной ввод данных. Второй — автоматическая загрузка данных из приборов для исследования скважин (Микон, Судос и т.д.). Возможна также автоматическая схема загрузки данных из глубинных геофизических приборов при условии наличия у последних цифрового блока хранения информации.

В программу предусмотрен ввод следующих данных: динамические и статические уровни скважины, давления (все виды устьевых давлений, глубинное с указанием глубины замера), результаты КВД, КВУ. В программе имеется возможность разложить один вид исследования на временные составляющие. Такая возможность необходима, например, при снятии кривых восстановления давления или уровня. В программе имеется генератор запросов и отчетов, с помощью которого можно производить мониторинг данных.
